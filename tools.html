Google data transfer (concept)

Transferring Google data (like Keep notes) between accounts is only possible through official Google APIs and export tools. Any solution must respect Google's Terms of Service and OAuth consent flow.

  • Sign in with source Google account and request read access.
  • Export items via Google APIs or Google Takeout.
  • Sign in with destination account and re-create notes using APIs.

Planned architecture

  1. Backend service (not on GitHub Pages) hosts OAuth flows and talks to Google APIs.
  2. Frontend (this site) only initiates flows and shows progress.
  3. All access tokens are stored server-side; the browser never sees secrets.
